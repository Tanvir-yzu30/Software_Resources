## Contributing
1. Fork this repository
2. Create a new branch (`git checkout -b feature/add-new-resource`)
3. Add/update resources (follow the existing format for consistency)
4. Commit your changes (`git commit -m 'Add [Tool/Library] to [Section]'`)
5. Push to the branch (`git push origin feature/add-new-resource`)
6. Open a Pull Request

### Guidelines
- Only add well-maintained, widely used resources
- Include direct links to official docs/downloads
- Keep descriptions concise and relevant
